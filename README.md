# AURA FASHION - AI Personal Fashion Stylist

AURA Fashion은 사용자의 전신 사진 실루엣과 키, 몸무게 정보를 바탕으로 개인맞춤형 패션 코디, 해시태그, 퍼스널 컬러 톤을 진단해주는 프리미엄 AI 스타일리스트 플랫폼입니다.

---

## ✨ 주요 기능 및 특징
1. **프리미엄 UI/UX**: 글래스모피즘(Glassmorphism) 카드, 커스텀 슬라이더, 스크롤 애니메이션 등 프리미엄 감성의 인터랙티브 디자인을 제공합니다.
2. **Precision Profiling**: 키와 몸무게 조절 및 드래그 앤 드롭 전신 사진 업로드를 통해 개인 실루엣 정보를 정확히 스캔합니다.
3. **AI 체형 분석 엔진**: 업로드된 정보와 사진을 구글 제미나이(Gemini 3.1 Flash-Lite) API가 고도로 분석하여 매력적인 코디 스타일 및 컬러 처방 리포트를 생성합니다.

## 🔐 Supabase Auth 연동
- **이메일 인증 기반 회원 관리**: Supabase Auth 연동을 적용하여 안전한 회원가입 및 로그인 프로세스를 구축했습니다.
- **분석 접근 제어**: 로그인된 회원에 한해 AI 스타일 분석을 허용하며, 미로그인 유저가 접근할 시 로그인 유도 모달이 팝업됩니다.

## 💡 트러블슈팅: 회원가입 시 `email rate limit exceeded` 오류 발생 시
수파베이스 이메일 회원가입 시 인증 메일 발송 횟수 제한으로 인해 회원가입이 되지 않는 경우, 아래 설정을 통해 즉시 해결할 수 있습니다.

1. **Supabase 대시보드** 접속 후 해당 프로젝트로 이동합니다.
2. **Authentication -> Providers -> Email** 메뉴로 진입합니다.
3. **Confirm email** 설정을 **비활성화(Disable)**합니다.
4. 설정을 저장하면 인증 절차와 한도 제약 없이 이메일/비밀번호만으로 즉시 가입 및 로그인이 가능합니다.
